# Turing’s Solstice Exchange — 90-second demo script

## 0:00–0:12 — Premise

“Welcome to **Turing’s Solstice Exchange**, a tiny browser game about surviving the longest trading day. The sun moves from dawn to sunset while Alan Turing’s fictional machine trains a new market maker.”

Show the home screen, five lesson trail, and locked market panel.

## 0:12–0:38 — Beginner lessons

“The game starts gently. Each lesson lets you make a decision before giving you the vocabulary: estimate fair value from noisy prints, choose a sensible bid and ask, see why spreads widen in riskier conditions, decode a Binary volatility clue, and mark inventory to PnL.”

Show Fair Value completion, Bid/Ask choice, and Volatility’s binary tape. Pause briefly on an unlocked glossary term.

## 0:38–1:10 — Market-maker mode

“After five lessons, the Solstice Exchange opens for eight rounds. Every round gives a small Turing signal—Binary, Caesar shift, or logic clue—then asks me to quote Tight, Balanced, or Wide. The simulation tracks fills, price movement, inventory, cash, and total PnL. The round recap explains what the decision taught, not just whether I won money.”

Show a signal decode, choose a spread, and a round summary with the execution ticker.

## 1:10–1:23 — Turing and AI note

“The Turing tribute is in the tape ciphers, machine logic, and code-like terminal aesthetic. The released build uses handcrafted static hints, so it works safely as a fully static GitHub Pages game. A disabled Gemini-provider interface is included for a future secured backend, not the public demo.”

Show the Decode help and Hint controls, then Settings.

## 1:23–1:30 — Close

“That’s Turing’s Solstice Exchange: a small, sunny way to learn why fair value, spread, volatility, inventory, and PnL belong in the same conversation. The repository and live demo links are below.”

Show Sunset Results and end on the title screen.
